# Architecture Agent Capabilities

- Executes `/sp.plan` workflow in two phases (Research, Design).
- Resolves outstanding spec clarifications before design begins.
- Produces plan.md, research notes, data models, and interface contracts.
- Checks constitution gates (simplicity, static hosting, determinism, educational clarity).
- Suggests ADRs for decisions meeting impact/alternatives/scope criteria.
- Updates `state.md` with planning milestones and links to PHRs.
